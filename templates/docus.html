Docus 是一个基于 UI Pro 文档模板 的主题。虽然视觉样式开箱即用，但你的重点应放在使用 Nuxt Content 提供的 Markdown 和 MDC 语法 来书写内容。

我们在所有 Nuxt 模块文档中都使用此主题，包括：

Nuxt Image

@nuxt/image 的文档

Nuxt Supabase

@nuxt/supabase 的文档

主要特点

此主题包含一系列旨在改善文档管理的功能：

  • 基于 Nuxt 3：利用最新的 Nuxt 框架以实现最佳性能。
  • 使用 Nuxt UI 和 Nuxt UI Pro 构建：集成了一整套全面的 UI 组件。
  • MDC 语法 通过 Nuxt Content：支持带有组件集成的 Markdown 以实现动态内容。
  • Nuxt Studio 兼容：以可视化方式编写和编辑内容，无需 Markdown 知识！
  • 自动生成侧边导航：根据内容结构自动生成导航。
  • 全文搜索：包含内置搜索功能以便于内容发现。
  • 优化排版：具有精致的排版以增强可读性。
  • 暗黑模式：为用户偏好提供暗黑模式支持。
  • 广泛功能：探索该主题以充分欣赏其能力。
