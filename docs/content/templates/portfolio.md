[Nuxt UI](https://ui.nuxt.com) 是一套高端组件集合，旨在帮助你在几分钟内创建美观且响应迅速的 Nuxt 应用。

Nuxt UI 团队致力于提供最佳的集成和定制体验，而 Studio 团队则提供与 Nuxt Studio 的完全兼容。

- **完全可定制**：可以从应用配置中更改任何组件的样式，也可以通过 ui 属性进行特定定制。
- **轻松书写 Markdown**：Nuxt UI 覆盖了 Nuxt Content 的 prose 组件，使其更出色，并新增了诸如 Callout、CodeGroup、Field 等组件。
- **漂亮的排版样式**：预配置并且样式匹配 Nuxt UI 组件和颜色的 Tailwind CSS 排版插件。
- **开箱即用的全文搜索**：Nuxt UI 配备了可直接使用的命令面板组件，无需再设置 Algolia DocSearch。
- **适用于所有的插槽**：每个组件都利用 Vue 的插槽强大功能，赋予你构建任何内容的灵活性。
- **响应式设计**：Nuxt UI 组件旨在结构化你的内容，天生响应式，能适应任何屏幕尺寸。
