[Nuxt UI](https://ui.nuxt.com) 是一组高级组件，旨在帮助你在几分钟内创建美观且响应迅速的 Nuxt 应用。

Nuxt UI 团队致力于提供最佳的集成和自定义体验，而 Studio 团队则提供与 Nuxt Studio 的全面兼容。

- **完全可定制**：通过你的应用配置更改任意组件的样式，或通过 ui 属性进行专门定制。
- **轻松编写 Markdown**：Nuxt UI 覆盖 Nuxt Content 的 prose 组件，使其更出色，还新增了 Callout、CodeGroup、Field 等组件。
- **优美的排版样式**：预配置了 Tailwind CSS 的排版插件，样式与 Nuxt UI 组件及颜色匹配。
- **开箱即用的全文搜索**：Nuxt UI 配备即用的命令面板组件，无需再配置 Algolia DocSearch。
- **全面支持插槽**：每个组件利用 Vue 插槽的强大功能，给予你构建任何内容的灵活性。
- **响应式设计**：Nuxt UI 组件旨在结构化你的内容，响应式设计，能够适应各种屏幕尺寸。
