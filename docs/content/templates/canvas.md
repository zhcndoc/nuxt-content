Canvas 作品集是一个完全可定制的国际化作品集模板，基于 Nuxt 和 Nuxt UI 构建，旨在帮助您轻松展示您的作品、推荐语和关键信息。该模板集成了 Nuxt Studio 模块，实现顺畅的编辑体验，同时利用 Nuxt Content 进行内容管理。Canvas 作品集在性能、灵活性和响应式方面表现优异，为开发者和创意人士提供坚实的基础。

- **现代组件与布局** – 包含内置组件。
- **Nuxt UI v3** – 利用预制的可定制 UI 组件。
- **支持 NuxtHub** - 几秒钟即可部署到 NuxtHub。
- **Tailwind CSS** – 美观且响应式的设计系统。
- **可用的联系表单** – 集成 Resend 轻松处理邮件。
- **多语言支持** – 由 Nuxt i18n 驱动。
- **SEO 准备就绪** – Open Graph 图片（Nuxt OG Image）及 Nuxt Robots 用于自动生成 robots.txt。
- **遵循最佳实践** – 自动生成网站地图，优化图片（Nuxt Image），以及 ESLint 配置（Nuxt 配置采用 Flat config）。
- **完全响应式** – 适配所有现代浏览器和设备。
- **极简且专业的设计** – 干净、优雅且易于定制。
