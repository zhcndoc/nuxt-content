一个轻量级的 Nuxt 主题，用于构建基于 Markdown 的网站，基于 Nuxt Content、TailwindCSS 和 Iconify

- 在 Markdown 页面中使用布局
- 享受元标签生成
- 可配置的 prose 组件
- 从页面自动生成导航
- 在明暗模式之间切换
- 访问来自 100+ 图标集的 100,000 个图标
- 使用 Shiki 高亮代码块
