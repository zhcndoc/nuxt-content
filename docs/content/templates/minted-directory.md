Minted 目录是一个高度可定制的模板，旨在快速构建成功的目录网站。

- 创建具有自定义风格/品牌的目录网站
- 使用 Markdown 管理列表
- SEO 优化
- 通过标签进行搜索和分类
- 博客页面

#right
  :::template-features
  ---
  features:
    - label: Nuxt 架构
      content: 利用 Nuxt 3 及其模块生态系统的全部强大功能。
    - label: 支持 Nuxt Studio
      content: 在 Nuxt Studio 模块中通过实时预览编辑您的内容。
    - label: Vue 组件
      content: 在内容中插入内置组件（或您自己的组件）。
    - label: 编写 Markdown
      content: 享受 Markdown 的简便和直观，并发现 MDC 语法。
    - label: 任何地方部署
      content: 可通过 Studio 一键部署，或者在 Vercel 或 Netlify 上零配置部署。支持静态生成、按需渲染（Node）或 CloudFlare workers 的边缘渲染。
    - label: 可扩展
      content: 完全自定义设计，或通过插槽添加组件—您可以让 Minted 目录成为您的专属。
